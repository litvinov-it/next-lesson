## Проект курса по NextJS 
Прошли на паре. Практика и основные фишки

- NextJS
- Prisma
- Zod
- NextAuth
- Tailwind
- DaisyUI
- bcrypt

## Запуск backend:
1. Записать свои данные в .env файл
2. Запустить докер:
```bash
docker compose up
```
3. Сделать миграцию
```bash
npx prisma migrate dev
```

## Запуск frontend:
```bash
npm run dev
```
